List of experiments for Microcontroller LAB

PART – A (Programming using simulator)

1. Study of addressing modes
  1. Different addressing modes used to access data like immediate,direct, indirect etc….(example program of Addition of two 8bit numbers)
2. Data Transfer instructions
  1. Block move 10 bytes data from 0X30-0X39 to 0X40-0X49 (without overlap)
  2. Block move - 10bytes of data from 0X30-0X39 to 0X35-0X39 (with overlap)
  3. Block exchange – 10bytes of data between 0X30-0X39 to 0X40-0X49
  4. Block move - 10bytes of data from 0X30-0X39 to 0X1000-0X1009 (Internal to external memory or vice versa)
  5. Smallest/Largest number in 10bytes of data from 0X30-0X39 (R3 – should store the smallest/largest number and R4 – should store address of the smallest/largest number)
  6. Search an element in the 10bytes of data from 0X30-0X39 (R3 =1,if element is found else R3=0)
  7. Sorting 10bytes of data from 0X30-0X39 in Ascending order
  8. Sorting 10bytes of data from 0X30-0X39 in Descending order
3. Arithmetic instructions:
  1. Add, sub, div, mul of 8 bit number with immediate data access
  2. Add and Sub of 16 bit numbers
  3. Square and Cube program
  4. To find LCM of given numbers
  5. To find HCF of given numbers
4. Boolean & Logical instructions :
  1. (CMP)Find 2’s complement of a number
  2. (ANL)Packed to Unpacked BCD(bit Masking)
  3. (ORL)Unpacked BCD to ASCII
  4. (XRL)ASCII to BCD
  5. (RLC)Count the number of positive and negative numbers in 10bytes of data from 0X30-0X39 (R3 should contain number of +ve element and R4 should contain number of –ve elements)
  6. (RRC)count the number of 1’s and 0’s in a byte(Parity detector) – (R3 should hold the result)
5. Counters
  1. HEX up/down counter - (Program should check value @R0=0X30, if 0X30=0 then up counter else down counter)
  2. BCD(00-99) up/down counter - (Program should check value @R0=0X30, if 0X30=0 then up counter else down counter)
6. Code converters
  1. Packed BCD to Hexadecimal and vice versa
7. Delay generation
  1. Without timers
8. Call and return instructions with port programming
  1. Logical or Delay loop using Call and return instructions

PART – B (Interfacing)

  1. LED interface with 8051
  2. Different waveform generation – Square, Sine, Triangle, RAMP etc using DAC interfacing module
  3. Stepper motor interfacing
  4. Interfacing 7segment display
