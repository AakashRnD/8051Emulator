8051EMULATOR

USER MANUAL

  • Launch Screen:

    The Launch screen appears with the start button. This provides the user with the basic information about the 8051Emulator application.

  • Workbench:

    1. After tapping the start button in the launch screen, the Workbench screen appears with a Microcontroller on the grid. This facilitates the user to connect the available interface circuits to the microcontroller and view the animated output based on assembly code written by the user.

    2. The sliding drawer at the left side of the workbench consists of three buttons for file handling operations:

      (a) New

      (b) Open

      (c) Save

  • Assembly Editor:

    The editor module facilitates the user to write the assembly code. This Editor module consists of the following:

      (a) Memory Usage

      (b) ROM Content

      (c) Log cat

      (d)Assemble Button: The Assemble button assembles the code written by the user.

      (e) Clear Button: The clear button clears all the contents in the Editor screen.

  • Internals 8051:

    The Internals8051 screen displays the values of the internal registers and special function registers after execution of the code in hexadecimal format.
